Installation

Composer

To install this package using composer, run: composer require mattcannon/Rightmove:0.1.*

Usage

To use the rightmove parser, create a new instance of the Parser class, passing in the path of the rightmove file. then call parseFile() on the instance - it will return a Collection of property objects.

<?php
use mattcannon\Rightmove\Parser;
$blmParser = new Parser('/path/to/rightmove.blm');
$collection = $blmParser->parseFile();

You can access the property attributes like so:

<?php
$property = $collection->first();
echo $property->displayAddress;
echo $property->price;
